Order management

Control and clarity for every purchase

My order has already shipped. Can I still cancel it?

Once shipped, cancellation isn’t possible. You can initiate a return after delivery. Use Main Menu → Returns for a swift, guided process.

Can I cancel my order?

Yes, if it hasn’t been delivered. Main Menu → Cancel → select item → reason → confirm. You’ll receive immediate confirmation.

How do I modify my order?

Contact support right away. Changes are only possible before shipping. After dispatch, use returns/exchanges instead.

What happens if I ordered the wrong item?

Cancel before shipping or return/exchange after delivery. Our team will help you switch to the right product quickly.

Can I add items to an existing order?

Not currently. Place a new order instead. We’re exploring bundled checkout to make this easier in the future.

Will I be charged for cancellations?

No fee if done before shipping. Once shipped, standard return policies apply. We keep the process fair and transparent.
